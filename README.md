# AI代码转换器 - 简易HTML生成工具

## 🌟 程序介绍

这是一个专门为**完全新手小白**设计的简易工具，帮助你将AI生成的HTML/CSS/JS代码转换为可直接运行的HTML文件。

无论你是从ChatGPT、文心一言还是其他AI工具获取代码，这个工具都能帮你快速检查代码完整性，并生成可以直接打开运行的HTML文件。

## ✨ 功能特点

### 📱 移动端适配
- ✅ 完全支持手机和平板等移动设备

### 🛠️ 核心功能
1. **代码输入** - 提供大文本框，方便粘贴AI生成的代码

   - HTML标签结构检查

3. **一键生成HTML** - 快速将代码转换为可运行的HTML文件

5. **清晰的操作指引** - 适合完全新手使用

## 🚀 使用方法

### 📋 基础使用步骤

1. **打开工具**
   - 找到 `ai-code-converter.html` 文件
   - 双击文件即可在浏览器中打开
   - 或右键点击文件，选择「打开方式」→ 选择你的浏览器（如Chrome、Edge、Safari等）

2. **粘贴代码**
   - 将AI生成的完整代码复制粘贴到文本框中
   - 确保代码包含完整的HTML结构（从 `<html>` 到 `</html>`）

3. **检查代码**
   - 点击「检查代码完整性」按钮
   - 等待系统自动检查代码
   - 查看检查结果，确认代码是否完整

4. **生成HTML文件**
   - 在「文件名」输入框中输入你想要的文件名（如 `my-app`、`calculator` 等）
   - 点击「生成HTML文件」按钮
   - 浏览器会自动下载生成的HTML文件

5. **运行HTML文件**
   - 打开你的「下载」文件夹
   - 找到刚才生成的HTML文件（文件名以 `.html` 结尾）
   - 双击文件即可在浏览器中运行
   - 或右键点击文件，选择「打开方式」→ 选择浏览器

### 📱 移动端使用方法

1. **在手机上打开工具**
   - 将 `ai-code-converter.html` 文件发送到手机（可通过微信、QQ、邮件等方式）
   - 在手机上下载并打开该文件
   - 系统会自动在手机浏览器中打开

2. **在手机上操作**
   - 点击文本框，粘贴AI生成的代码
   - 点击「检查代码完整性」按钮
   - 输入文件名
   - 点击「生成HTML文件」按钮
   - 手机会自动下载HTML文件

3. **在手机上运行生成的文件**
   - 打开手机的「文件管理」应用
   - 找到「下载」文件夹
   - 找到生成的HTML文件
   - 点击文件即可在手机浏览器中运行

## ❓ 常见问题

### Q: 为什么我的代码检查不通过？
A: 可能是代码缺少必要的HTML结构，比如 `<html>`、`<body>` 标签，或者标签没有正确闭合。请检查AI生成的代码是否完整。

### Q: 生成的HTML文件在哪里？
A: 文件会下载到你的默认下载文件夹中。你可以在浏览器的下载历史中找到它。

### Q: 生成的HTML文件无法打开怎么办？
A: 请确保文件扩展名是 `.html`，如果不是，可以手动修改文件名，添加 `.html` 后缀。

### Q: 我可以在哪些设备上使用这个工具？
A: 只要设备上有现代浏览器（如Chrome、Edge、Safari、Firefox等），就可以使用这个工具，包括：
   - Windows电脑
   - Mac电脑
   - 安卓手机/平板
   - iPhone/iPad

### Q: 这个工具需要联网吗？
A: 不需要！这个工具是纯本地运行的，完全不需要联网，保护你的隐私。
